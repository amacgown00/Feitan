Numbers

  • + = addition
  • - = subtraction
  • / = division
  • * = multiplication
  • ** = exponent (2 ** 2) = 4
  • Float — a number with a decimal point (dividing any 2 numbers will produce a float).
  • CONSTANT = unchanging variable, must be in ALL CAPS.

Lists - set of items in a particular order

In square brackets with elements separated by commas. 

Ages = [3, 15, 20]

    orchid_genera = ["phragmipedium", "phalaenopsis", "cattleya"]

Elements can be called by index.
print(orchid_genera[2].title())

Cattleya -1 = last item in the list. Append, a method that adds an element to the end of the list 
orchid_genera.append("dendrobium")
Insert - method to add an element at a certain position 
orchid_genera.insert(1, "catasetum")
Del - statement to remove an element at a certain position 
del orchid_genera[0]
Remove - method to remove an element by value when you don't know its index. 
orchid_genera.remove("phalaenopsis")
