Git

1. Check current status


                git status
  • Shows new, modified, and deleted files.
  • Tells you if the working tree is clean.

2. Create folders


            mkdir -p 01-folder
  • -p makes parent folders if missing.

3. Create or overwrite a file

touch 01-folder/doc.md

Or write content directly from the terminal:

echo 'Same as it ever was' > 01-folder/doc.md

4. Delete files

rm 01-folder/doc.md

Or delete multiple files at once:

rm 01-folder/*.tmp

5. Stage changes (including deletes)

git add .

Stages all new, modified, or deleted files.
Optional: stage only tracked changes and deletions:

git add -u

6. Commit

git commit -m "Notes you can write about the change"

7. Pull remote changes

git pull --no--rebase

Merges remote changes into your local branch.
Finalize merge:

git commit
